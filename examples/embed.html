Tayo360 Widget — Embed Example

This page demonstrates the simplest possible embed. Open the floating button in the bottom-right to interact with the AI assistant.

How it works: the widget self-injects all required HTML and CSS into the page at load time. No frameworks, no build step, no HTML edits to your site beyond a single <script> tag.

Minimal embed

<script src="tayo360-widget.js" defer></script>

With configuration

<script>
  window.Tayo360Config = {
    webhookUrl: 'https://your-n8n.example.com/webhook/tayo360-lead',
    brandName: 'Tayo360',
    botName: 'Tayo360 Assistant',
    primaryColor: '#0a2540',
    accentColor: '#1a73a8',
    autoOpenAfterMs: 0  // set e.g. 8000 to auto-open after 8 seconds
  };
</script>
<script src="tayo360-widget.js" defer></script>

Trigger from any element on the page

Add data-tayo360-open to any element to open the widget on click:

<button data-tayo360-open>Talk to our AI assistant</button>

Or call the JavaScript API

Tayo360.open();
Tayo360.close();
Tayo360.toggle();
